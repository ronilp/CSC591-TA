Install tensorflow 1.4.1
If you want to install cpu version :
pip install tensorflow==1.4.1
or
conda install tensorflow==1.4.1

For gpu version :
pip install tensorflow-gpu==1.4.1
or
conda install tensorflow-gpu==1.4.1

Install keras 2.2.2
conda install keras==2.2.2
pip install keras==2.2.2

This project is to be done in a python version >= 3.6
